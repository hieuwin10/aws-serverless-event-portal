Về Luồng Dữ Liệu & Giải Pháp Cache Miễn Phí

Biểu đồ trên thể hiện luồng hoạt động chính: Người dùng tải nội dung tĩnh từ Amplify, lấy Token xác thực từ Cognito, và gọi API qua API Gateway.

Để thay thế Redis tốn kém, AWS Lambda được cấu hình để truy xuất DynamoDB như một lớp Cache trung gian cực nhanh trước khi đẩy lệnh xuống Primary DB (RDS). Nhờ tính năng TTL, dữ liệu cache tự xóa khi hết hạn mà không tốn phí, giữ toàn bộ kiến trúc lõi trong vùng AWS Free Tier vĩnh viễn.
